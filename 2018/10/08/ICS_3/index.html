Posted on 2018-10-08 | In ICS

程序的机器级表示

  • Complex instruction set computer (CISC)
    储存占用低，CPU复杂
  • Reduced instruction set computer (RICS)
    内存冗余，关注时间效率，CPU设计简单

Intel 8086->386(32位)->Pentium->Pentium Pro(CICS,硬件转换为RICS)->Pentium 4E(64位)->Core 2(multi core)-> i7
移动平台 ARM(RICS)

Definitions

  • Architecture(also ISA,instruction set architechure)
    • Machine Code
    • Assembly Code
  • Microarchitechture Implementation of ISA

Example ISAs:

  • Intel x86, IA32, Itanium, x86-64
  • ARM
  • RISC V:open-sourse

Assembly/Machine Code View

CPU

  • PC:Program counter
    • Address of next instruction
    • Called RIP(x86-64)
  • Register file
    • heavily used data
  • Condition codes
    • store status information about most recent arithmatic or logic operation used for conditional branching

Memory

Turning C into Object Code

  • C program p.c -Compiler->
    • *dest = t;
  • Asm program p.s(gcc -Og -S p.c) -Assembler->
    • movq %rax (%rbx)
  • Object program p.o -Linker->
    • 0x40059e: 48 89 03
  • program

objdump

Assembly Characteristics

Data Type

  • Integer of 1,2,4,8 bytes
  • Floating point of 4,8,10 bytes
  • Code
  • No aggregte type

Operation

  • arithmetic function
  • transfer data
  • transfer control
    • jump
    • conditional branches

Moving Data

movq Sourse, Dest

Memory Addressing Modes

  • (R) Mem[Reg[R]]
  • D(R) Mem[Reg[R]+D]
  • D(Rb,Ri,S) Mem[Reg[Rb]+S*Reg[Ri]+D]
    • S=1,2,4,8
    • Ri can’t be %rsp

Processor State

  • Temporary data(%rax…)
  • Location of runtime stack(%rsp…)
  • current code control point(%rip…)
  • CF ZF SF OF…

Condition Codes

Single bit registers

  • CF carry flag(unsigned)
  • ZF zero flag
    • set 1 when the result is 0
  • SF sign flag
    • set 1 when <0
  • OF overflow flag(signed)
    • (a>0&&b>0&&t<0)||(a<0&&b<0&&t>=0)

Not set by leaq instruction

Explicit Setting

  • cmpq Src1, Src2
    • Src2-Src1
  • testq Src1, Src2
    • Src2&Src1
    • testq %rax, %rax 判断是否为0

SetX Instructions

set low-order byte of destination to 0 or 1 based on conditional code, does not alter the remain 7 bytes

  • equal
    • sete ZF 相等
    • setne ~ZF 不等
  • sign
    • sets SF 负数
    • setns ~SF 非负数
  • signed compare
    • setl SF^OF 小于(less)
    • setle (SF^OF)|ZF 小于或等于
    • setg 大于(grater)
    • setge
  • unsigned compare
    • setb CF 小于(below)
    • setbe CF|ZF 小于或等于
    • seta 大于(above)
    • setae

movzbl %al, %eax会把高4字节设为0

Jumping

JX Instruction

  • Unconditional
    • jmp
  • equal
    • je
    • jne
  • …

Conditional Expression Translation

条件转移不能准确判断分支去向

Conditional Moves

条件传送不破坏流水线，每条语句都执行，但是当分支过于复杂，或者部分分支不安全(野指针)，或者分支有连带效应(x*=7)时会出现问题
