Memory Hierarchy

Radom-Access Memory(RAM)

磁带不是RAM
断电信息丢失

  • SRAM (Static RAM)

    用于Cache

  • DRAM (Dynamic RAM)

    信息以电容？方式储存，通电时也需要刷新，储存密度更高，成本更低，用于主内存

    • SDRAM (Synchronous DRAM)
    • DDR SDRAM (Double data-rate synchronous DRAM)

Nonvolatile Memories

断电也不会消失

  • ROM (Read-only memory)
    • programmed during production

    • Firmware programs: BIOS, controller for disks, network cards, graphics accelerators, security subsystems…

    • Solid state disks

    • Disk caches

  • PROM (Programmable ROM)
    • can be programmed once
  • EPROM (Eraseable PROM)
    • can be bulk erased (UV, X-Ray)
  • EEPROM (Electrically eraseable PROM)
  • EEPROMs (Flash memory)

    可在线修改，不用拆下来修改

platters with two surfaces

  • Record density
  • Track density

    磁道径向密度

  • Earal density

CPU把需要硬盘需要执行的指令、block编号、目标内存地址传给disk controller，然后硬盘执行操作把需要的数据传给内存，CPU干别的事去了，直到数据被写进内存，发送中断指令到CPU。
